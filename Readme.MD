# QP Assignment

Grocery Booking API

## Installation

Clone the project.

```bash
git clone https://github.com/mansoorshakir53/qp-assessment.git
```

Go to project.

```bash
cd qp-assessment
```

Run the project.

```bash
sudo docker-compose up --build -d
```
Down the project.

```bash
sudo docker-compose down --remove-orphans
```

## Usage

Import QP ASSIGNMET.postman_collection.json file in postman.
